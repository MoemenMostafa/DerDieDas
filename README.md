# Artikel Welt (DerDieDas)

Artikel Welt is a mobile application built with Ionic and Angular, designed to help users master German articles (Der, Die, Das) through interactive learning.

## Features

- **Interactive Learning**: Practice German articles with a user-friendly interface.
- **Text-to-Speech**: Listen to the pronunciation of words to enhance learning.
- **Progress Tracking**: Keep track of your learning journey with visual progress indicators.
- **Cross-Platform**: Built using Ionic and Capacitor for a seamless mobile experience.
- **Multi-language Support**: Integrated translations for a global audience.

## Tech Stack

- **Framework**: [Ionic](https://ionicframework.com/) & [Angular](https://angular.io/)
- **Native Bridge**: [Capacitor](https://capacitorjs.com/) & Cordova
- **Styling**: Sass (SCSS)
- **Utilities**: 
  - `canvas-confetti` for celebrations
  - `ng-circle-progress` for visual tracking
  - `ngx-translate` for internationalization

## Getting Started

### Prerequisites

- Node.js (v12+ recommended)
- Ionic CLI (`npm install -g @ionic/cli`)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MoemenMostafa/DerDieDas.git
   cd DerDieDas
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application locally:
   ```bash
   ionic serve
   ```

## Development Commands

- `npm start`: Runs the app in development mode.
- `npm run build`: Builds the app for production.
- `npm run test`: Runs unit tests.
- `npm run lint`: Lints the source code.
- `npm run android`: Syncs the project with Capacitor and builds the Android debug APK.
